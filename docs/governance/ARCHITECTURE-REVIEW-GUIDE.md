# Architecture Review Guide

Architecture review is required for package additions, dependency rule changes, CI governance changes, significant tooling changes, and all future implementation-phase boundaries.

## Required Evidence

- ADR reference or proposed ADR
- Compliance matrix requirement IDs
- dependency-cruiser impact assessment
- TypeScript project reference impact
- Security and operations implications
- Rollback or migration plan where applicable

## Decision Outcomes

- Approved
- Approved with required controls
- Rejected due to constitutional conflict
- Deferred pending ADR or additional evidence
