# Governance Template

## Governance Item

Name and identifier.

## Authority

Identify the constitutional authority, ADR, or policy source.

## Scope

State exactly what repository areas, teams, and workflows are governed.

## Required Controls

List automated checks, human reviews, evidence artifacts, and escalation paths.

## Non-Compliance Handling

Describe failure behavior, remediation expectations, and exception process.

## Review Cadence

Define periodic review and owner responsibilities.
